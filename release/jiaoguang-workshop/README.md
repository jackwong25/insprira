# 交广新媒工坊 · insprira v1.1

济南交通广播 · 新媒体内容创作平台 · 个人便携版

---

## 快速开始

### 前提条件
- **Node.js ≥ 20**：[下载安装](https://nodejs.org/zh-cn/)
- Windows 10/11

### 启动
1. 解压 `jiaoguang-workshop-v1.1.zip` 到任意目录
2. 双击 `start.bat`
3. 浏览器自动打开 `http://127.0.0.1:8080`
4. ⚠️ **命令行窗口不能关**，关了服务就停了。不用时关掉即可

### 首次登录
- 账号：`admin`
- 密码：`123456`
- ⚠️ 登录后请立即在「设置 → 账户与安全」修改密码

---

## 配置（必做）

登录后点击左下角「设置」→「维护 .env」，填写以下字段后保存，重启服务：

| 字段 | 说明 | 示例 |
|------|------|------|
| **RedFox API Key** | 热点数据/违禁词/封面 | 从 [redfox.hk](https://redfox.hk) 注册 |
| **LLM API Key** | AI 创作调用 | DeepSeek/OpenAI 等 |
| **LLM Base URL** | API 端点 | `https://api.deepseek.com/v1` |
| **LLM 模型** | 模型名 | `deepseek-chat` |
| **知识库加密密钥** | 加密 Notion 凭据 | 用 `openssl rand -hex 32` 生成 |

### 个人 LLM Key（可选）
在「维护 .env」中往下翻，填写 **个人 LLM API Key / Base URL / 模型**。
- 填了 → 用你自己的 Key
- 不填 → 用上方的系统共享 Key
- 适合多人用一台机器，各用自己的额度

## 功能导航

### 📊 热榜
左侧「热榜」→ 查看全网实时热点
- 基础：抖音、小红书、公众号
- 扩展：通过 Skill 中心绑定 AI 公众号/B站/快手/视频号等
- 日榜、7天/14天趋势分析

### ✍️ AI 创作（核心功能）
左侧「AI 创作」→ 内容生成与改写

#### 三种工作模式

| 模式 | 用途 | 校对策略 |
|------|------|----------|
| **重构** | 粘贴原文 → AI 扩展结构和打磨 | 严格比对原文，不编造新事实 |
| **创作** | 输入主题/关键词 → AI 从零写 | 防幻觉，允许常识扩展 |
| **改写** | 粘贴原文 → 换风格 | 不增删任何信息 |

#### 操作步骤
1. 在热榜中点击「发送到创作」，或手动粘贴原文/输入关键词
2. 选择目标平台（公众号/小红书/知乎/抖音等）
3. 选择篇幅（简洁/标准/详细 — **仅创作模式生效**）
4. 可选：选择「风格档案」（需先在「我的账号」提炼）
5. 点击按钮开始
6. 右侧生成标题、前言、正文
7. 使用「违禁词体检」检测违规内容
8. 使用「AI 封面」生成配图
9. 点「复制成稿」或「保存到草稿箱」

#### 平台篇幅对照表（创作模式）

| 平台 | 简洁 | 标准 | 详细 |
|------|-----|------|------|
| 公众号 | ~500字 | ~1200字 | ~2500字 |
| 知乎 | ~500字 | ~1500字 | ~3000字 |
| 小红书 | ~100字 | ~300字 | ~600字 |
| 抖音 | ~50字 | ~150字 | ~300字 |
| 快手 | ~80字 | ~200字 | ~400字 |
| 视频号 | ~80字 | ~200字 | ~400字 |
| B站 | ~300字 | ~800字 | ~1500字 |

### 🗂️ 草稿箱
左侧「草稿箱」→ 管理创作内容
- 创作页点「保存到草稿箱」
- 草稿箱中查看所有已保存草稿
- 点「继续编辑」→ 整篇文章进入素材框，可直接修改
- 再次保存会**更新原草稿**（不产生重复）
- 点垃圾桶图标删除

### 💡 选题生成
左侧「选题」→ 基于本地热榜数据自动生成选题
- 需先配置 LLM
- 按平台/来源筛选
- 选中可「发送到创作页」

### 👥 账号追踪
左侧「账号追踪」→ 竞品/参考账号
- 添加公众号/抖音/小红书账号
- 自动同步作品列表
- 生成趋势图表
- 提炼风格档案供创作参考

### 📚 知识库
左侧「知识库」→ 接入外部知识源
- Obsidian 本地 Vault
- Notion Database
- WeRss 公众号 RSS

### 🤖 Agent 对话（可选）
左侧「Agent 对话」→ 调用本地 AI CLI

需先安装对应 CLI：
```bash
npm i -g @openai/codex          # Codex
npm i -g @anthropic-ai/claude-code  # Claude Code
npm i -g hermes-agent            # Hermes
```

## 常见问题

### Q: 启动报错"Node.js not found"
A: 从 https://nodejs.org/zh-cn/ 下载安装 LTS 版（≥20）

### Q: 登录按钮没反应
A: 检查是否开了 VPN/代理。关闭 VPN 或使用浏览器无痕模式

### Q: 创作模式字数不对
A: 确认三点：① 选了「创作」模式 ② 选了正确篇幅 ③ LLM 配置正确

### Q: 如何分享给同事？
A: 把整个 `jiaoguang-workshop` 文件夹压缩发过去。每人可在「维护 .env」配自己的 LLM Key

### Q: 数据存在哪？
A: 所有数据存在本机 `app/cache.db`（SQLite），不会上传到任何服务器

### Q: 如何更新版本？
A: 下载新版本 zip，用其中 `app/` 目录替换旧 `app/` 目录。`.env` 和 `cache.db` 保留不动
